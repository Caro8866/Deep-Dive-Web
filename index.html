Text Wrapping

In classic text editing and document creating programs like Adobe InDesign and Microsoft Word, you are able to wrap text around images, other text and more. I was curious if the same could be achieved using HTML & CSS and it can using CSS shapes and shape-outside property.

Here you will be introduced to the many ways you can wrap text:
  • Basic wrapping around a square using float property
  • Text wrap around a circle
  • Text wrap around a polygon
  • Text wrap around a png file
  • Pull quotes

Text wrap around a circle

To wrap sections of text around a circle all you need to use is the shape-outside property.

Short step-by-step guide:

  1. Set display flex and flex-direction: column on the overall container containing both the image and text.
  2. Then set a float: left (or right depending on your case) on the circle.
  3. Border-radius: 50% to ensure perfect circle.
  4. Shape-outside: circle(50%) is then used to create a shape outline in the form of a circle.

Scrolling Text

Scrolling text can easily and quickly turn a boring and static website, fun and dynamic with minimal effort.

Here you will be introduced to the many ways you can create moving & scrolling text:

  • Horizontal scrolling
  • Vertical scrolling
  • Circular scrolling

All these methods use CSS instead of the deprecated <marquee> tag. I will focus on the CSS implementation of scrolling text as it is supported by all browsers.

Horizontal Scrolling

To achieve horizontal scrolling of text you need to use CSS animation property alongside the @keyframes rule. To make text scroll horizontally you need to use translateX() in order to determine the position of the text at the beginning and end of the animation which will be put on it. These two starts and ending points will determine where the text moves to and from.

JAVASCRIPT EXPLANATION

  1. The code starts by finding the html element with id="circleText" on the page.
  2. The code then changes its innerHTML to be a string of all the characters in that text, and it splits up each character into an array.
  3. Next, it maps over each character in turn and turns them into a span tag with a rotation angle of 8.2 degrees for each one.
  4. Finally, it joins all these spans together to make one big string again.
